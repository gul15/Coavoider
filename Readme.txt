In order to see the objects in the world correctly, you must modify the paths with "dupin"
in the coavoider.world file according to your computer name.

Dependencies:
Built-in ros libraries
pip install SpeechRecognition
pip install pipwin
pipwin install pyaudio

How to Run:
roslaunch coavoider coavoider.launch
rosrun coavoider controller.py

Commands:
("garbage" or "medical") and (1,2,3,4)
Examples -> garbage 1, medical 3
